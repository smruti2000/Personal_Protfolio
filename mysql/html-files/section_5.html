Introduction to CRUD

  • CRUD is an acronym for
               Create
               Read
               Update
               Delete
  • CRUD operations are basic data manipulation for database.
  • We've already learned how to perform create (i.e. insert), read (i.e. select), update and delete operations in previous sections.

SELECT

  • The SELECT statement is used to pull information from a table
  • The general format is:
                mysql> SELECT what_to_select FROM which_table;
  • For Example:
    • mysql> SELECT * FROM pet;
      O/P: It gives the all data of the "PET" table.
    • mysql> SELECT name,date FROM pet;
      O/P: It gives only the name and date of the "PET" table.

WHERE

  • By "WHERE" we can add the conditions for our statementes.
  • The general format is:
                mysql> SELECT what_to_select FROM which_table WHERE conditions_to_satisfy ;
  • For Example:
    • mysql> SELECT * FROM pet WHERE age=10;
      O/P: It gives the all data which cat have age-10 of the "PET" table.
    • mysql> SELECT name,date FROM pet WHERE name='piu';
      O/P: It gives only the name and date which cat name="piu" of the "PET" table.

ALIASES(AS)

  • Temperary change of the table header
  • But in the original table name can not be changed.
  • For Example:
    • mysql> SELECT name AS 'Pet_Name' FROM pet;
      O/P: It gives the 'NAME' data of the "PET" table As Header name 'Pet_Name'.
